# 课程讲义：第十节 - 结语与最佳实践

## 1. 核心能力回顾

通过本课程，我们掌握了 Claude Code 在以下方面的应用：

- **代码库探索** (Exploration)
- **自动化测试** (Testing)
- **代码重构** (Refactoring)
- **故障调试** (Debugging)

## 2. 提升效率的三大支柱

- **清晰的指令** (Clear Instructions)：明确你想要达成的目标。
- **丰富的上下文** (Context Clarification)：让模型理解当前的开发环境。
- **精准的文件引用** (Relevant Files)：通过 `@` 符号指向具体文件，减少无效搜索。

## 3. 项目长期维护

- **CLAUDE.md**：这是 Claude Code 的“记忆中枢”。
  - 记录代码规范 (Style Guides)。
  - 记录常用的测试命令或构建流程。
  - 记录项目特有的业务逻辑。

## 4. 扩展性：MCP 服务器

- **跨平台协作**：连接 Figma (设计稿)、Playwright (UI 测试) 等。
- **自定义工具**：根据团队需求接入更多 MCP 资源。
