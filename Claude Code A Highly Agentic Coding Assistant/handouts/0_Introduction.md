## 什么是 Claude Code？

Claude Code 是一款高度代理化（Agentic）的编程助手。

- **演进过程**：AI 辅助编程经历了从简单的问答、代码补全（如 GitHub Copilot）到如今具备高度自主性的“代理”阶段。
- **核心特性**：它不仅能提供建议，还能在数分钟内自主处理复杂任务。开发者甚至可以并行运行多个 Claude 实例来协作处理代码库的不同部分。

## 核心使用最佳实践

高效使用 Claude Code 的关键在于提供**清晰的上下文**：

- **指定相关文件**：明确告诉 Claude 需要处理哪些文件。
- **描述明确的功能需求**：清晰界定你想要实现的功能和特性。
- **利用工具扩展**：通过 MCP（Model Context Protocol）服务器扩展其能力，使其能够与外部工具和生态系统无缝衔接。

## 课程实战案例

本课程将通过三个具体示例来深入实践：

1. **RAG 聊天机器人**
   - 实现从前端到后端的全栈开发。
   - 包括代码重构、测试编写，以及利用 GitHub 集成处理 PR 和修复 Issue。
   - 掌握规划、思考模式、并行会话和内存管理等高级功能。

2. **Jupyter Notebook 数据分析**
   - 针对电子商务数据进行探索性分析。
   - 重构 Notebook 并移除冗余代码，将其转化为功能强大的 Web 应用仪表盘。

3. **Figma 原型转代码**
   - 使用 Figma MCP 服务器直接导入设计稿。
   - 迭代测试并代理化地构建出高性能的前端应用程序。

## Claude Code 的底层原理与优势

- **架构极简**：不同于许多需要复杂预索引的工具，Claude Code 仅依靠少量基础工具（如模式搜索、目录列举、文件读取、正则匹配）。
- **代理化理解**：它通过自主阅读代码并在 `CLAUDE.md` 中记录关键信息来理解代码库，而非依赖语义嵌入。
- **本地与安全**：由于不需要对整个代码库进行索引或上传，代码可以完全保留在本地，具备更高的安全性。
