# 课程讲稿：第十节 - 结语

恭喜你完成了这门课程的学习！

到目前为止，你已经学习了如何使用 Claude Code 来探索、测试、重构以及调试代码库。

为了充分发挥 Claude Code 的威力，请确保你：

1. 给它**清晰的指令**。
2. 明确当前的**上下文信息**。
3. 指向代码库中**相关的特定文件**。

此外，请务必将你的代码库规范（codebase rules）以及任何你希望 Claude Code 记住的项目信息添加到 **`CLAUDE.md`** 文件中。

你还可以考虑扩展 Claude Code 的能力，将其连接到如 Playwright 和 Figma 等 **MCP 服务器**。

感谢你与我一起开启这段旅程。我迫不及待地想看到你使用 Claude Code 构建出怎样的杰作！
